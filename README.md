# Социальная сеть вкусов фильмов (MVP)

**Идея:** найти людей с похожим вкусом, а не просто рекомендовать фильмы.  

**Основные фичи:**
- Личная [витрина](showcase.md) фильмов пользователя  
- Импорт оценок через TMDb  
- Автодозагрузка постеров и метаданных  
- Нахождение похожих пользователей (косинусное сходство)  
- Рекомендации от соседей  
- Разделение фильмов на любимые, ненавистные и спорные  

**Технологии:**
- Telegram Bot + канал для хранения постеров  
- SQLite (стартер)  
- Python + NumPy для расчёта сходства
